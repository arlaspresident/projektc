# ExpenseApp

## Beskrivning

ExpenseApp är en konsolapplikation utvecklad i **C# (.NET)** som hjälper användaren att få överblick över sina utgifter på ett enkelt sätt.
Syftet med projektet är att skapa ett lokalt verktyg som inte kräver inloggning eller databas, utan sparar information direkt i en **JSON fil** på datorn.



## Struktur

Applikationen är uppdelad i tre huvuddelar:

* **Domain** – datamodellen för en utgift
* **Services** – logik och hantering av lagring
* **UI** – konsolgränssnittet med meny


## Kör applikationen

1. Klona repot:

   ```bash
   git clone (https://github.com/arlaspresident/projektc.git)
   ```
2. Gå in i mappen:

   ```bash
   cd expense-app
   ```
3. Kör applikationen:

   ```bash
   dotnet run
   ```
